# UnityPokemon

A demo recreation of **Pokémon Platinum** built in **Unity 6**.

This is a personal, non-commercial fan project exploring how the systems and feel of the original Generation IV Pokémon games translate into a modern Unity engine. It is not affiliated with, endorsed by, or sponsored by Nintendo, Game Freak, or The Pokémon Company.

## About

The goal of this project is to learn and experiment by rebuilding the core gameplay loop of Pokémon Platinum from the ground up in Unity, including:

- Grid-based overworld movement
- Tile-based maps and encounters
- Turn-based battle system
- Pokémon stats, moves, and type matchups
- Menus, dialogue, and UI

This is a **demo / work-in-progress**, not a complete game.

## Stack

- **Engine:** Unity 6
- **Languages:** C#, ShaderLab, HLSL
